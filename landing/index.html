Project Notes

Human-Centric Design

This project was built with a focus on usability and nostalgia. I wanted to capture the feel of the original handheld devices while using modern web technologies.

Technological Stack

Built using React (via Babel CDN for simplicity), the PokéAPI for live data, and custom CSS for that authentic "handheld" look.
